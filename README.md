# Excel_Dashboard-BankLoanReport

# Overview

This project presents an interactive Bank Loan Analysis Dashboard built using Microsoft Excel. It analyzes loan data to track performance, assess risk, and understand borrower behavior through dynamic KPIs and visualizations.
# Problem Statement

Financial institutions deal with large volumes of loan data, making it difficult to:

Monitor key KPIs like loan applications, funding, and repayments
Track Month-to-Date (MTD) and Month-over-Month (MoM) trends
Identify high-risk (bad) loans
Analyze borrower segments across multiple dimensions
Make quick, data-driven decisions

# Solution

Developed a multi-dashboard Excel solution to transform raw data into actionable insights:

📊 Summary Dashboard: KPI tracking & Good vs Bad loan analysis
🌍 Overview Dashboard: Trends by time, geography, and borrower attributes
📋 Details Dashboard: Granular data exploration with filters
📈 Key Features
Total Loan Applications tracking
Total Funded Amount & Amount Received
MTD & MoM performance analysis
Good vs Bad Loan segmentation (risk analysis)
Interest Rate & Debt-to-Income (DTI) insights
Loan distribution by:
State
Term
Employment Length
Purpose
Home Ownership
🛠️ Tools & Techniques
